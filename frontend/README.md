# Frontend (Angular)

## Requirements
- Node.js 18+
- npm 11+

## Install and Run
```bash
npm install
npm start
```

Dev server URL: `http://localhost:4200`

## Build
```bash
npm run build
```

## Notes
- Uses Angular standalone components and route guards.
- Uses Angular Material and Bootstrap.
- API calls target `http://localhost:8080/api`.
- Requests are sent with credentials and CSRF header for mutating operations.
